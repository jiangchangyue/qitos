# qitos-zoo Migration Staging

This directory is temporary staging material for seeding a separate `qitos-zoo` repository. It is not part of the QitOS Python package and should not be treated as canonical framework source.

`qitos-zoo` is the home for product-grade and showcase-grade applications built with QitOS, including:

- `qitos-coder`: a Claude Code-inspired coding agent.
- `qitos-cyber-agent`: a PentAGI-inspired cybersecurity agent for controlled security research workflows.
- `experimental`: app candidates that need more product ownership before becoming maintained zoo apps.

Generic abstractions should move down into QitOS only after multiple independent zoo apps need them.
