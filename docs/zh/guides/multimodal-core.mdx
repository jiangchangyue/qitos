# 多模态内核与桌面入门

QitOS v0.5 已经不只是多模态基础设施。

它已经把这一条主线接完整了：

- 截图优先的多模态输入
- `DesktopEnv`
- 官方 `desktop` 基准测试家族
- `openai_cua_agent.py` 基线
- qita 可视化检查

## qita 在 v0.5 里能看到什么

- 截图时间线
- 回放截图预览
- 基础动作叠加层
- 接地元数据
- 评估器重试信息

## 当前边界

v0.5 仍然**不承诺**：

- 完整的官方 OSWorld 对等
- v0.6 级别的完整可视化回放深度
- 所有提供商的全面多模态对齐

它承诺的是一件事：

**一条完整可信的桌面入门路径**
