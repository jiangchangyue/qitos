# 构建法律咨询 Agent

本指南展示如何使用 QitOS 构建一个中文法律咨询 Agent，集成通义千问 (Qwen) 模型和 DashScope 向量检索。

## 前置条件

- 安装 QitOS：`pip install qitos[models]`
- 获取 DashScope API Key：[阿里云控制台](https://dashscope.console.aliyun.com/)

## 基本架构

法律咨询 Agent 需要三个核心组件：

1. **AgentModule** — 定义 Agent 的行为和状态
2. **向量知识库** — 存储法律法规和案例
3. **审查机制 (Critic)** — 确保回答合规且不越界

## 实现步骤

### 1. 定义状态

```python
from dataclasses import dataclass, field
from typing import List, Dict
from qitos.core.state import StateSchema

@dataclass
class LegalConsultState(StateSchema):
    """法律咨询 Agent 状态。"""
    query: str = ""
    relevant_articles: List[Dict] = field(default_factory=list)
    legal_analysis: str = ""
    disclaimer_added: bool = False
```

### 2. 创建 AgentModule

```python
from qitos.core.agent_module import AgentModule

class LegalConsultAgent(AgentModule[LegalConsultState, Any, Any]):
    def init_state(self, task: str, **kwargs):
        return LegalConsultState(task=task, max_steps=15)

    def build_system_prompt(self, state):
        return (
            "你是一个专业的法律咨询助手。你的职责是：\n"
            "1. 根据用户问题检索相关法律条文\n"
            "2. 提供客观的法律分析\n"
            "3. 在回答末尾添加免责声明\n"
            "注意：你不能提供具体法律建议，只能提供法律信息参考。\n"
            f"用户问题：{state.task}"
        )
```

### 3. 配置向量知识库

```python
from qitos.kit.embedding import DashScopeEmbedder
from qitos.kit.memory import VectorMemory

embedder = DashScopeEmbedder(
    model="text-embedding-v3",
    api_key="your-dashscope-api-key",
)

knowledge = VectorMemory(embedder=embedder, top_k=5)

# 索引法律条文
knowledge.append({"content": "《民法典》第一千一百六十五条：行为人因过错侵害他人民事权益造成损害的，应当承担侵权责任。"})
```

### 4. 添加合规审查

```python
from qitos.engine.critic_decorator import critic

@critic(name="legal_disclaimer", score=0.8)
def legal_disclaimer_critic(state, decision, results):
    """确保回答包含免责声明。"""
    if state.disclaimer_added:
        return "continue"
    return (
        "retry",
        "回答缺少免责声明",
        "重要：你必须在回答末尾添加以下免责声明："
        "\"以上内容仅供参考，不构成法律建议。如需专业法律意见，请咨询持证律师。\""
    )
```

### 5. 运行 Agent

```python
from qitos.engine.engine import Engine
from qitos.models.openai_compatible import OpenAICompatibleModel

model = OpenAICompatibleModel(
    model="qwen-plus",
    api_key="your-dashscope-api-key",
    base_url="https://dashscope.aliyuncs.com/compatible-mode/v1",
)

agent = LegalConsultAgent(tools=[...])
engine = Engine(agent=agent, model=model)
engine.add_critic(legal_disclaimer_critic)

result = engine.run("劳动合同解除有哪些法定情形？")
print(result.state.legal_analysis)
```

## 最佳实践

- **免责声明强制**：使用 Critic 确保每次回答都包含免责声明
- **向量检索精度**：使用 DashScopeEmbedder 获取更准确的中文语义向量
- **多轮对话**：利用 CompactHistory 管理长对话上下文
- **合规边界**：在系统提示中明确划定 Agent 的能力边界
