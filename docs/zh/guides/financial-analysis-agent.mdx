# 构建金融分析 Agent

本指南展示如何使用 QitOS 构建一个金融数据分析 Agent，集成智谱 GLM 模型和向量检索。

## 前置条件

- 安装 QitOS：`pip install qitos[models]`
- 获取智谱 API Key：[智谱开放平台](https://open.bigmodel.cn/)

## 基本架构

金融分析 Agent 的关键要素：

1. **多数据源整合** — 通过工具读取财务报表、市场数据
2. **风险审查** — 使用 Critic 确保分析客观、不构成投资建议
3. **结构化输出** — 使用 JSON 协议确保输出格式一致

## 实现步骤

### 1. 定义状态

```python
from dataclasses import dataclass, field
from typing import List, Dict, Optional
from qitos.core.state import StateSchema

@dataclass
class FinancialAnalysisState(StateSchema):
    """金融分析 Agent 状态。"""
    target_company: str = ""
    financial_data: Dict = field(default_factory=dict)
    analysis_results: List[str] = field(default_factory=list)
    risk_factors: List[str] = field(default_factory=list)
    disclaimer_added: bool = False
```

### 2. 创建 AgentModule

```python
from qitos.core.agent_module import AgentModule

class FinancialAnalysisAgent(AgentModule[FinancialAnalysisState, Any, Any]):
    def init_state(self, task: str, **kwargs):
        return FinancialAnalysisState(
            task=task,
            target_company=task,
            max_steps=20,
        )

    def build_system_prompt(self, state):
        return (
            "你是一个专业的金融数据分析助手。\n"
            "你的职责是：\n"
            "1. 收集和分析目标公司的财务数据\n"
            "2. 识别关键风险因素\n"
            "3. 提供客观的数据分析报告\n"
            "注意：你不能提供投资建议，只能提供数据分析参考。\n"
            f"分析目标：{state.target_company}"
        )
```

### 3. 配置智谱向量

```python
from qitos.kit.embedding import ZhipuEmbedder
from qitos.kit.memory import VectorMemory

embedder = ZhipuEmbedder(
    model="embedding-3",
    api_key="your-zhipu-api-key",
)

knowledge = VectorMemory(embedder=embedder, top_k=5)
```

### 4. 添加风险审查

```python
from qitos.engine.critic_decorator import critic

@critic(name="financial_risk_guard", score=0.8)
def financial_risk_critic(state, decision, results):
    """确保分析报告包含风险提示和免责声明。"""
    # 检查是否有投资建议倾向
    analysis_text = " ".join(state.analysis_results)
    investment_keywords = ["建议买入", "推荐投资", "必涨", "稳赚"]
    for keyword in investment_keywords:
        if keyword in analysis_text:
            return (
                "retry",
                f"检测到投资建议倾向：{keyword}",
                "请修改：移除投资建议表述，只保留客观数据分析。"
                "在报告末尾添加风险提示和免责声明。"
            )

    if not state.disclaimer_added:
        return (
            "retry",
            "缺少风险提示",
            "请在分析报告末尾添加：\"本报告仅供参考，不构成投资建议。"
            "投资有风险，决策需谨慎。\""
        )

    return "continue"
```

### 5. 运行 Agent

```python
from qitos.engine.engine import Engine
from qitos.models.openai_compatible import OpenAICompatibleModel

model = OpenAICompatibleModel(
    model="glm-4-flash",
    api_key="your-zhipu-api-key",
    base_url="https://open.bigmodel.cn/api/paas/v4",
)

agent = FinancialAnalysisAgent(tools=[...])
engine = Engine(agent=agent, model=model)
engine.add_critic(financial_risk_critic)

result = engine.run("某上市公司")
```

## 使用 JSON Decision 协议

金融分析 Agent 推荐使用 `json_decision_v1` 协议，确保输出结构化：

```python
from qitos.protocols import get_protocol

protocol = get_protocol("json_decision_v1")
# GLM 默认使用此协议
```

## 最佳实践

- **风险审查**：使用 Critic 过滤投资建议表述
- **结构化输出**：使用 JSON 协议确保结果可解析
- **数据验证**：对财务数据输入进行边界检查
- **审计日志**：使用 Tracing 记录分析过程
