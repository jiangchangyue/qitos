# 国产模型生态概览

QitOS 原生支持多个国产大语言模型家族，通过统一的 `FamilyPreset` 机制实现零配置切换。

## 支持的模型家族

### DeepSeek

| 属性 | 值 |
|------|-----|
| FamilyPreset ID | `deepseek` |
| 默认协议 | `json_decision_v1` |
| 适配方式 | OpenAI Compatible |
| 推荐模型 | `deepseek-chat`, `deepseek-reasoner` |
| 上下文窗口 | 128K |

DeepSeek 使用 JSON Decision 协议输出工具调用，适合需要深度推理的场景。

### 通义千问 (Qwen)

| 属性 | 值 |
|------|-----|
| FamilyPreset ID | `qwen` |
| 默认协议 | `json_decision_v1` |
| 适配方式 | OpenAI Compatible (DashScope) |
| 推荐模型 | `qwen-plus`, `qwen-turbo`, `Qwen3-8B` |
| 上下文窗口 | 128K |
| 原生工具调用 | 支持 |

通义千问支持原生工具调用 (Native Tool Call)，可通过 `native_tool_call_preferred=True` 启用。

### 智谱 GLM

| 属性 | 值 |
|------|-----|
| FamilyPreset ID | `glm` |
| 默认协议 | `json_decision_v1` |
| 适配方式 | OpenAI Compatible |
| 推荐模型 | `glm-4`, `glm-4-flash` |
| 上下文窗口 | 128K |

智谱 GLM 使用 JSON Decision 协议，适合结构化输出场景。

### MiniMax

| 属性 | 值 |
|------|-----|
| FamilyPreset ID | `minimax` |
| 默认协议 | `minimax_tool_call_v1` |
| 适配方式 | OpenAI Compatible |
| 推荐模型 | `MiniMax-M2.5` |
| 上下文窗口 | 128K |
| 原生工具调用 | 支持 |

MiniMax 使用专有的 `<minimax:tool_call>` 标记，支持多工具并行调用。

## 向量嵌入适配

QitOS 提供国产模型的原生 Embedder 适配：

### DashScopeEmbedder

```python
from qitos.kit.embedding import DashScopeEmbedder

embedder = DashScopeEmbedder(
    model="text-embedding-v3",  # 1024维
    api_key="your-dashscope-api-key",
)
```

支持模型：`text-embedding-v3` (1024维), `text-embedding-v2` (1536维)

### ZhipuEmbedder

```python
from qitos.kit.embedding import ZhipuEmbedder

embedder = ZhipuEmbedder(
    model="embedding-3",  # 2048维
    api_key="your-zhipu-api-key",
)
```

支持模型：`embedding-3` (2048维), `embedding-2` (1024维)

## 协议架构

```
国产模型 → OpenAI Compatible API → FamilyPreset → ModelProtocol → Parser
                                                          ↓
                                              tool_schema_renderer
                                              prompt_renderer
                                              contract_renderer
```

每个模型家族有独立的 `ModelProtocol` 定义渲染方式：

- **JSON Decision** (`json_decision_v1`) — DeepSeek、Qwen、GLM
- **MiniMax Tool Call** (`minimax_tool_call_v1`) — MiniMax 专有
- **ReAct Text** (`react_text_v1`) — 通用回退

## 切换模型

```python
from qitos.harness._types import FamilyPreset, build_protocol_for_preset

# 使用 DeepSeek
protocol = build_protocol_for_preset(preset=DEEPSEEK_PRESET)

# 切换到 Qwen
protocol = build_protocol_for_preset(preset=QWEN_PRESET)
```

## 添加新模型家族

1. 在 `qitos/harness/presets.py` 中定义 `FamilyPreset`
2. 如需新协议，在 `qitos/protocols.py` 中注册 `ModelProtocol`
3. 添加 Embedder（如有 OpenAI 兼容端点）
4. 编写测试和示例

参见 [添加 Family Preset](/zh/guides/add-a-family-preset) 获取详细步骤。
