# @function_tool 迁移指南

QitOS 提供了两种工具装饰器：`@tool`（标记式）和 `@function_tool`（构建式）。推荐使用 `@function_tool`。

## 对比

| 特性 | @tool | @function_tool |
|------|-------|----------------|
| 类型推断 | 基础 | 丰富（从类型注解推断） |
| needs_approval | 不支持 | 支持 |
| read_only | 不支持 | 支持 |
| timeout_s | 不支持 | 支持 |
| 返回类型 | 原函数 | FunctionTool 实例 |
| schema 质量 | 简单 | 详细参数描述 |

## 迁移步骤

### 1. 替换导入

```python
# 旧
from qitos.core.tool import tool

# 新
from qitos.core.function_tool_decorator import function_tool
```

### 2. 替换装饰器

```python
# 旧
@tool(name="shell", description="Execute shell command")
def shell(self, command: str) -> str:
    return subprocess.run(command, shell=True, capture_output=True).stdout

# 新
@function_tool(name="shell", description="Execute shell command", needs_approval=True)
def shell(self, command: str) -> str:
    return subprocess.run(command, shell=True, capture_output=True).stdout
```

### 3. 添加标记

- `needs_approval=True` — 有副作用的工具（写文件、执行命令、网络请求）
- `read_only=True` — 只读工具（读取文件、搜索、列表）

### 4. 移除 ToolPermission

`@function_tool` 不使用 `permissions=ToolPermission(...)` 参数。

```python
# 旧
@tool(name="shell", permissions=ToolPermission(needs_approval=True))

# 新
@function_tool(name="shell", needs_approval=True)
```

## 在类方法上使用

`@function_tool` 支持在类方法上使用，自动处理 `self` 绑定：

```python
class MyToolSet:
    @function_tool(name="read_file", read_only=True)
    def read_file(self, path: str) -> str:
        return open(path).read()

    @function_tool(name="write_file", needs_approval=True)
    def write_file(self, path: str, content: str) -> None:
        open(path, 'w').write(content)
```

## FunctionTool 描述符协议

`FunctionTool` 实现了 Python 描述符协议（`__get__`），在类实例上访问时自动绑定 `self`：

```python
toolset = MyToolSet()
# toolset.read_file 是一个绑定的 FunctionTool
# toolset.read_file.run(path="test.txt") 会自动传入 self
```

## 自定义工具示例

```python
from qitos.core.function_tool_decorator import function_tool

@function_tool(
    name="web_search",
    description="Search the web for information",
    needs_approval=True,      # 网络访问需要审批
    timeout_s=30.0,           # 30 秒超时
    max_retries=2,            # 最多重试 2 次
)
def web_search(query: str, max_results: int = 5) -> list[str]:
    """Search the web and return URLs."""
    ...
```

## 迁移注意事项

1. **工具注册**: `FunctionTool` 实例可以直接注册到 `ToolRegistry`
2. **DelegatingTool**: 如果工具已经通过 `FunctionTool` 包装，`DelegatingTool` 会自动检测并复用
3. **命名空间**: 在 `register_toolset` 中，工具的 `spec.name` 会被复制以避免命名空间前缀污染
