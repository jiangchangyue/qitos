# QitOS 社区

欢迎加入 QitOS 社区！我们致力于构建一个开放、包容的 AI Agent 研究者社区。

## 交流渠道

### 微信群

搜索微信号 **qitos-community** 或扫描二维码添加社区助手，邀请入群。

群内讨论主题：
- QitOS 框架使用和最佳实践
- 国产模型适配和调优
- Agent 模式和架构设计
- 基准测试和评测方法

### 飞书群

加入飞书群获取：
- 框架更新公告
- 线上技术分享通知
- 核心开发者 Office Hour

搜索群号 **QitOS-Community** 加入。

## 贡献方式

### 贡献 Agent 应用

参考 [Zoo 贡献指南](https://github.com/Qitor/qitos/blob/main/plans/qitos_zoo_migration/CONTRIBUTING.md) 提交你的 Agent 应用。

### 贡献模型适配

如果你在适配新的国产模型，欢迎提交 FamilyPreset 和 Embedder 实现。

### 贡献文档和示例

中文文档和示例始终欢迎改进。特别是：
- 行业应用示例（法律、金融、医疗等）
- 国产模型最佳实践
- 教程和指南的中文翻译

## 行为准则

参与社区需遵守 [Code of Conduct](https://github.com/Qitor/qitos/blob/main/CODE_OF_CONDUCT.md)。

核心原则：
- 尊重每位社区成员
- 建设性讨论，避免攻击性言论
- 欢迎不同水平的贡献者
- 保持讨论与项目相关

## 核心团队

| 成员 | 方向 |
|------|------|
| QitOS Team | 框架核心开发 |
| Community Maintainers | 社区运营和文档 |

## 社区活动

- **月度技术分享**：每月最后一个周五，分享 Agent 技术最新进展
- **贡献者峰会**：季度性线上活动，表彰贡献者并规划下季度方向
- **Workshop**：不定期线下/线上实战工作坊

关注 [GitHub Discussions](https://github.com/Qitor/qitos/discussions) 获取活动通知。
