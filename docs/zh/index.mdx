QitOS 是面向智能体（agent）研究者的类 PyTorch 风格框架。你可以在同一个 `AgentModule + Engine` 内核上原型化方法、运行基准测试（benchmark），并用内建 `qita` 观察长时运行轨迹（trajectory）。

## 两种编写方式

QitOS 提供两种顶层的智能体编写方式：

- 研究优先（research-first）：手写提示词（prompt）、解析器（parser）、协议（protocol，模型输出的格式契约）、传输层（transport，与模型 API 通信的客户端）和工具面（tool surface），直接研究与优化内核（kernel，执行内核）。
- 预设优先（preset-first）：从家族预设（family preset，预配置的模型家族默认值）和预设工具构建器出发，快速得到稳定基线（baseline）与多模型切换能力。

这两种方式都是一等公民。

## 快速启动

<Steps>
  <Step title="安装 QitOS">
    ```bash
    pip install "qitos[models]"
    ```
  </Step>
  <Step title="设置提供商配置">
    ```bash
    export OPENAI_API_KEY="sk-..."
    export OPENAI_BASE_URL="https://api.siliconflow.cn/v1/"
    export QITOS_MODEL="Qwen/Qwen3-8B"
    ```
  </Step>
  <Step title="运行最小编程智能体">
    ```bash
    qit demo minimal
    ```
  </Step>
  <Step title="用 qita 查看运行轨迹">
    ```bash
    qita board --logdir runs
    ```
    打开 [http://127.0.0.1:8765](http://127.0.0.1:8765)，查看追踪记录、回放（replay）和导出（export）结果。
  </Step>
</Steps>
