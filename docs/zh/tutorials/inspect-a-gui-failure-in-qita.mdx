# 在 qita 中检查 GUI 故障

桌面运行失败不能只看解析器。

你需要回答：

- 模型看到了什么
- 它点击或输入了什么
- 定位是否准确
- 评估器（critic，可批准、停止或重试当前步骤）是否拒绝了弱动作

```bash
qita replay --run ./runs/<run_id>
```

重点检查：

- 可视化时间轴
- 回放截图预览
- 动作（action，标准化工具调用）叠加层
- 失败标记
