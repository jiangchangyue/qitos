QitOS 博客是文档之外的另一条内容线。

文档主要解释 QitOS **是什么、怎么用**；博客则更适合回答：

- 为什么框架会这样设计
- 哪些方法在真实运行中有效
- 哪些基准测试或长时运行中的失败值得被总结
- 提示词、解析器、工具、记忆与可观测性在实践里如何相互作用

## 这里会写什么

我们希望后续博客主要围绕这些主题扩展：

- **设计笔记**：为什么框架采用现在的结构
- **实战复盘**：长时编码与审计智能体的经验与坑
- **基准测试观察**：GAIA、Tau-Bench、CyBench 的实践结论
- **可观测性**：`qita` 如何把追踪记录变成研究产物
