## 黄金预设

前五个家族（Qwen、Kimi、MiniMax、`gpt-oss`、Gemma 4）是**黄金预设**——QitOS 团队在每次发版时验证得最充分的配置。当你刚上手或需要跑基准对比时，优先选择黄金预设。

后五个（OpenAI、Anthropic、Gemini、DeepSeek、GLM）是**兼容预设**。它们为各自的家族提供了正确的协议和上下文默认值，但随着专用适配器的加入可能会演进。

## 正确理解这张表

这张表不是模型排行榜。

它表达的是：当你在 QitOS 里选择某个家族预设（预配置的模型家族默认值）时，框架默认会采用什么模型侧契约。

如果你需要改某一项，请显式覆盖：

- 用 `build_model_for_preset(protocol=...)` 覆盖协议
- 用 `build_model_for_preset(tool_delivery=...)` 覆盖工具交付方式
- 用 `preset.override()` 扩展预设——参见[家族预设](/zh/concepts/family-presets)

不要把整个示例分叉成另一套实现。
